طراحی سایت

متناسب با نوع شغل شما و هدف شما از ایجاد سایت ، هرآنچه که نیاز هست را با به روز ترین تکنولوژی ،امنیت بالا و کیفیت مطلوب به شما ارائه میدهم.

Html

HTML مخفف Hyper Text Markup Language بوده و در فارسی به آن زبان نشانه‌ گذاری ابرمتن می‌گویند. دقت کنید که HTML یک زبان برنامه نویسی نیست.

Bootstrap

بوت استرپ یک فریم ورک سمت کلاینت است. یک چارچوب مدرن برای توسعه سریع و قدرتمند وب‌سایت که شامل ابزارهای طراحی براساس HTML و CSS می‌باشد.

JavaScript

جاوا اسکریپت زبان برنامه‌نویسی پویا و مبتنی بر شی (Object-based) است که هم سمت کاربر و هم سمت سرور، برای ایجاد تعمال با صفحات وب به کار می‌رود.

React

reactjs به عنوان یک ابزار جاوااسکریپتی با هدف ایجاد وبسایت‌های SPA شناخته می‌شود که شما با استفاده از آن می‌توانید خیلی راحت چنین وبسایت‌های ایجاد کنید.

ارسال پیام به من

در این بخش شما میتوانید پیام یا درخواست خود را برای من ارسال کنید. سعی میکنم در اسرع وقت پاسخگوی پیام‌های شما عزیزان باشم.
همچنین شما میتونید بصورت «ناشناس یا شناس» نظر، انتقاد، پیشنهاد و هر آنچه در دلتون هست برای من بفرستید. تشویق های شما باعث دلگرمی من و انتقادات شما باعث پیشرفت من میشه پس منتظرتون هستم.
